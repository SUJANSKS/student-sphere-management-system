# 🎓 Student Sphere Management System

A web-based application to manage student profiles, job postings, and application tracking using Spring Boot.

---

## 🚀 Features

### 👨‍🎓 Student
- Create and update academic profile
- View available job listings
- Apply for jobs
- Track application status (⏳ Pending / ✔ Approved / ❌ Rejected)

### 🧑‍💼 Recruiter
- Post new jobs
- View all applications
- Approve or reject applications

### 🛡️ Admin
- View system dashboard
- Monitor users, jobs, and applications

---

## 🧠 Key Functionalities

- ✅ MVC Architecture (Controller → Service → Repository)
- ✅ Session-based login system
- ✅ Application status management
- ✅ Duplicate job application prevention
- ✅ Dynamic UI using Thymeleaf
- ✅ Clean and responsive UI

---

## 🛠️ Tech Stack

- **Backend:** Spring Boot (Java)
- **Frontend:** Thymeleaf, HTML, CSS
- **Database:** MySQL
- **Build Tool:** Maven
- **Version Control:** Git & GitHub
